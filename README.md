# Tech-Blog

## Description
This is a blog site for the technical issues. Users are able to use this app by creating an account. The application uses MySql for database functionality, Handlebars for HTML templating, Bcrypt for security and password encryption. There are other packages like Sequelize, Express, MySql2 and Dotenv. 

## Installation
1. Clone the repo
1. Run `npm i` to install all the packages and dependencies!
1. Run `node seeds/index.js` feed the database with prepared data
1. Run `node server.js`

## Usage
This a simple app for anyone who are interested to leave a comment under-net others post or post their own topics.
